1. 데이터 타입이 일치해야 한다.
→ 데이터 타입을 변환하는 과정이 필요하다




* 다음 세 문장에서 컴파일 오류가 발생하는 이유는?

int num1 = 10000000000;		// num에 저장 불가 

long num2 = 10000000000;	// 상수가 먼저 표현되므로

float num3 = 12.45;		// 12.45는 double형 상수 


접미사 L은 long형으로 형 변환,
접미사 F는 float형으로 형 변환을 의미함.
